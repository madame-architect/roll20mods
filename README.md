# Roll Table
## Run it (with the Ammunition Table, otherwise import your own)

Assuming your handout is named Ammunition and the table is in Notes. 

!mdtbl --handout|Ammunition --table|Ammunition --mode|replace --showplayers|false --field|notes

This will create a table named Ammunition.

## If you pasted into GM Notes instead:

!mdtbl --handout|Ammunition --table|Ammunition --mode|replace --showplayers|false --field|gmnotes
